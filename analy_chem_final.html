Chromophore & Auxochrome
Chromophore

Q&A เน้นข้อสอบ 🔥 ออกแน่
Q: เงื่อนไขการใช้ Beer's Law?
A: 1) แสง monochromatic  2) สารละลายเจือจาง  3) สารละลายเนื้อเดียวกัน
Q: $n \rightarrow \\pi^*$ + polar solvent → shift แบบใด? ทำไม?
A: Blue shift — ตัวทำละลายมีขั้วเกิด H-bond กับ lone pair → stabilize ground state → ต้องใช้พลังงานมากขึ้น → $\\lambda$ สั้นลง
Q: $\\pi \rightarrow \\pi^*$ + polar solvent → shift แบบใด? ทำไม?
A: Red shift — excited state มีขั้วมากกว่า ground state → ตัวทำละลายจับ excited state → stabilize excited state → พลังงานลด → $\\lambda$ ยาวขึ้น
Q: conjugation เพิ่มขึ้น → ผล?
A: $\\lambda_{max}$ เพิ่มขึ้น (red shift) + $\varepsilon$ เพิ่มขึ้น (hyperchromic) เพราะ energy gap ระหว่าง ground–excited state ลดลง
Q: ทำไมเซลล์แก้วใช้กับ UV ไม่ได้?
A: เนื้อแก้วดูดกลืน UV เองทำให้ $P_0$ ที่แท้จริงลดลง → $A$ ที่วัดได้สูงกว่าจริง → ต้องใช้ Quartz หรือ Silica แทน
Q: Grating 3 แบบต่างกันอย่างไร?
A: 1) Transmission grating — ไม่ค่อยใช้แล้ว  2) Reflection/Diffraction grating — สะท้อนแสงจากพื้นผิวที่เป็นร่อง (นิยมมากที่สุด)  3) Echelle grating — ระยะห่างระหว่างร่องใหญ่มาก มุมรับแสงสูงมาก ความละเอียดสูง
Q: Photomultiplier ทำงานอย่างไร?
A: แสงตกกระทบ photocathode → ปล่อย $e^-$ → $e^-$ วิ่งชน dynode แต่ละตัว → ผลิต $e^-$ เพิ่มเป็นทวีคูณ → กระแสสุดท้ายขยายมากกว่าเดิมมาก → sensitivity สูง
ตัวอย่างโจทย์คำนวณ 🧮 ฝึกคำนวณ
โจทย์ 1: สารละลายมี $A = 0.450$ วัดด้วยเซลล์กว้าง 1.00 cm ความเข้มข้น $2.00 \times 10^{-4}$ mol/L จงหา $\varepsilon$
จาก $A = \varepsilon b c$
$\varepsilon = \dfrac{A}{bc} = \dfrac{0.450}{1.00 \times 2.00 \times 10^{-4}}$
$\varepsilon = 2250 \text{ L·mol}^{-1}\text{·cm}^{-1}$
โจทย์ 2: ถ้า $\varepsilon = 1500$, $b = 2.00$ cm ต้องการ $A = 0.600$ ต้องใช้ความเข้มข้นเท่าใด?
$c = \dfrac{A}{\varepsilon b} = \dfrac{0.600}{1500 \times 2.00}$
$c = 2.00 \times 10^{-4}$ mol/L
โจทย์ 3: $A = 0.300$ จงหา %Transmittance
$A = -\\log T \Rightarrow T = 10^{-A} = 10^{-0.300} = 0.501$
%$T = 50.1\%$
โจทย์ 4: สาร X ใน MeOH (polar) วัดได้ $\lambda_{max} = 340$ nm เมื่อเปลี่ยนเป็น hexane (nonpolar) $\lambda_{max}$ เปลี่ยนเป็น 360 nm บอก transition type และ shift
polar solvent → nonpolar: ถ้า $\lambda$ เพิ่ม (340→360) = red shift เมื่อเปลี่ยนไป nonpolar
polar solvent ทำให้ $\lambda$ สั้นลง → transition นี้เป็น $\pi \rightarrow \pi^*$ (polar solvent ทำ red shift แต่ nonpolar ทำ blue shift สำหรับ $n \rightarrow \pi^*$)
$\pi \rightarrow \pi^*$ transition — polar solvent → red shift, nonpolar → blue shift
โจทย์ 5: สาร A มี $\varepsilon = 10000$, สาร B มี $\varepsilon = 200$ ทั้งคู่ความเข้มข้น $10^{-4}$ M, $b = 1$ cm วัด A รวม = 1.020 ถ้า A แท้ของ A = 1.000 จงหา A ที่ stray light ทำให้เกิด error กี่ %?
A จริงของ A = $\varepsilon \times b \times c = 10000 \times 1 \times 10^{-4} = 1.000$
A ที่วัดได้ = 1.020 แต่จริงๆ = 1.000 → error = (1.020−1.000)/1.000 × 100%
%error = 2.0% (เกิดจาก stray light หรือ background absorption)

Self-quenching vs Self-absorption
• Self-quenching = ความเข้มข้นสูงมาก → โมเลกุลชนกัน → ส่งพลังงานให้กัน → ไม่ปล่อยแสง
• Self-absorption = โมเลกุลดูดกลืนแสง fluorescence ที่ปล่อยออกมาเอง → ความเข้มลด
• ทั้งสองอย่างทำให้กราฟ $I_F$ vs $c$ เบี่ยงเบนจากเส้นตรงเมื่อความเข้มข้นสูงเกินไป
Quenching vs Inner-Filter Effect
ประเภทกลไกผล
Static quenchingสารเจือปน (ground state) จับกับโมเลกุล → เกิด non-fluorescent complexลด $\Phi_F$
Dynamic quenchingสารเจือปนชนกับโมเลกุลใน excited state → ดึงพลังงานออกลด $\Phi_F$
Inner-filter ($I_a$)สารตัวอย่างดูดกลืนแสงที่ใช้กระตุ้น → แสงไม่ถึงโมเลกุลลดความเข้มโดยตรง
Inner-filter ($I_b$)สารตัวอย่างดูดกลืนแสง emission ก่อนถึง detectorลดความเข้มโดยตรง

Q&A เน้นข้อสอบ 🔥 ออกแน่
Q: Fluorescence vs Phosphorescence ต่างกันอย่างไร?
A: Flu = $S_1 \rightarrow S_0$ เร็ว ($10^{-8}$–$10^{-5}$ s) หยุดทันทีเมื่อตัดแสง / Phos = $T_1 \rightarrow S_0$ ช้า ($10^{-4}$–$10$ s) ผ่าน ISC → ยังปล่อยแสงต่อหลังตัดแสง
Q: ISC คืออะไร? เกี่ยวข้องกับอะไร?
A: $S_1 \rightarrow T_1$ โดย spin กลับทิศ (forbidden transition) เกิดช้า → นำไปสู่ Phosphorescence ถ้า ISC มาก → Fluorescence น้อยลง เพราะโมเลกุลแยกออกไปเป็น triplet state แทน
Q: $\Phi_F$ สูงต้องเป็นอย่างไร?
A: $k_F$ มาก (transition แบบ $\\pi \rightarrow \\pi^*$) + $k_{ISC}$ น้อย + โครงสร้าง rigid + ไม่มี $e^-$-withdrawing group บน benzene
Q: ทำไม Fluorescence ต้องใช้ Monochromator 2 ตัว?
A: ตัวแรกเลือก excitation $\\lambda$ / ตัวที่สองเลือก emission $\\lambda$ โดยวาง 90° เพื่อป้องกันแสงกระตุ้นตกกระทบ detector โดยตรง — ถ้า UV-Vis ต้องการวัดเฉพาะ absorption จึงใช้แค่ 1 ตัว
Q: Self-quenching vs Self-absorption ต่างกันอย่างไร?
A: ทั้งคู่เกิดเมื่อความเข้มข้นสูงเกิน — Self-quenching: โมเลกุลในสถานะกระตุ้นชนกับโมเลกุลอื่น → ส่งพลังงานให้กัน ไม่ปล่อยแสง / Self-absorption: โมเลกุลดูดกลืนแสง emission ที่ตัวเองปล่อยออกมา
ตัวอย่างโจทย์คำนวณ 🧮 ฝึกคำนวณ
โจทย์ 1: สาร A มี $\Phi_F = 0.80$, $k_F = 2.0 \times 10^8$ s⁻¹ จงหา $k_{IC} + k_{ISC} + k_Q[Q]$ (รวมทุก deactivation นอกจาก $k_F$)
\[ \Phi_F = \frac{k_F}{k_F + k_{others}} \Rightarrow 0.80 = \frac{2.0 \times 10^8}{2.0 \times 10^8 + k_{others}} \]
$k_{others} = 2.0\times10^8(1/0.80 - 1) = 2.0\times10^8 \times 0.25$
$k_{others} = 5.0 \times 10^7$ s⁻¹
โจทย์ 2: สาร B มี $\lambda_{excitation} = 350$ nm แต่ $\lambda_{emission} = 420$ nm อธิบาย Stokes shift และคำนวณผลต่างพลังงาน
Stokes shift = $\lambda_{em} - \lambda_{ex}$ = 420 − 350 = 70 nm
พลังงานสูญเสีย = $hc(1/\lambda_{ex} - 1/\lambda_{em})$ = สูญเสียไปกับ vibrational relaxation
Stokes shift = 70 nm → พลังงาน emission ต่ำกว่า excitation เสมอ (ตาม Stokes law)
โจทย์ 3: เมื่ออุณหภูมิเพิ่มจาก 25°C เป็น 50°C ความเข้ม fluorescence ลด 30% อธิบายทำไม
อุณหภูมิ↑ → การชนระหว่างโมเลกุลมากขึ้น → External conversion / Dynamic quenching เพิ่ม
→ $k_Q[Q]$ เพิ่ม → ตัวส่วนใน $\Phi_F$ เพิ่ม → $\Phi_F$ ลด → ความเข้มลด 30%
อุณหภูมิ↑ → quenching↑ → $\Phi_F$↓ → ความเข้ม fluorescence↓
โจทย์ 4: ทำไม Fluorescence ถึงวัด emission ที่ 90° กับแสงกระตุ้น?
ถ้าวัดตรงแนวแสงกระตุ้น → แสงกระตุ้นที่ไม่ถูกดูดกลืนจะตรงไปยัง detector โดยตรง → background สูงมาก → sensitivity ต่ำ
วัดที่ 90° → แสงกระตุ้นไม่ถึง detector → วัดได้เฉพาะ fluorescence emission → S/N สูงกว่า UV-Vis มาก
90° = ลด background จากแสงกระตุ้น → sensitivity สูงกว่า UV-Vis 100–1000 เท่า
ตัวอย่างโจทย์คำนวณ 🧮 ฝึกคำนวณ
โจทย์ 1: สาร A มี $\Phi_F = 0.80$, $k_F = 2.0 \times 10^8$ s⁻¹ จงหา $k_{others} = k_{IC} + k_{ISC} + k_Q[Q]$
\[ 0.80 = \frac{2.0 \times 10^8}{2.0 \times 10^8 + k_{others}} \Rightarrow k_{others} = 2.0\times10^8 \times \frac{1-0.80}{0.80} \]
$k_{others} = 5.0 \times 10^7$ s⁻¹
โจทย์ 2: $\lambda_{ex} = 350$ nm, $\lambda_{em} = 420$ nm จงหา Stokes shift และอธิบาย
Stokes shift = 420 − 350 = 70 nm
สาเหตุ: โมเลกุลสูญเสียพลังงานไปกับ vibrational relaxation ก่อน emit → $\lambda_{em}$ ยาวกว่า $\lambda_{ex}$ เสมอ
Stokes shift = 70 nm
โจทย์ 3: เปรียบ sensitivity: Fluorescence วัด quinine sulfate ได้ที่ $10^{-9}$ g/mL ส่วน UV-Vis วัดได้ที่ $10^{-6}$ g/mL Fluorescence sensitive กว่ากี่เท่า?
sensitive กว่า = $10^{-6} / 10^{-9}$
Fluorescence sensitive กว่า UV-Vis 1000 เท่า เพราะวัดที่ background ต่ำกว่า (90°)
โจทย์ 4: ถ้าเติม quencher [Q] = 0.01 M ทำให้ $\Phi_F$ ลดจาก 0.80 เป็น 0.50 จงหา $k_Q$ (กำหนด $k_F + k_{IC} + k_{ISC} = 3.0 \times 10^8$ s⁻¹)
$0.50 = k_F/(3.0\times10^8 + k_Q \times 0.01)$ และรู้ว่า $k_F = 0.80 \times 3.0\times10^8 / (1) = 2.4\times10^8$
$0.50 = 2.4\times10^8/(3.0\times10^8 + 0.01k_Q)$ → $3.0\times10^8 + 0.01k_Q = 4.8\times10^8$
$k_Q = 1.8 \times 10^{10}$ L·mol⁻¹·s⁻¹

สูตรสำคัญ — Retention & Separation
Retention Factor ($k'$) — วัดว่าสาร elute ช้าแค่ไหน
$$k' = \frac{t_R - t_M}{t_M} = \frac{m_s}{m_m}$$
Linear Flow Velocity
$$\\bar{u} = \frac{L}{t_R}$$
Selectivity Factor ($\\alpha \\geq 1$ เสมอ — คำนวณจากสารที่ออกช้า / เร็ว)
$$\\alpha = \frac{K_B}{K_A} = \frac{k'_B}{k'_A} = \frac{t_r(B) - t_M}{t_r(A) - t_M}$$
Resolution ($R_s \\geq 1.5$ = แยกสมบูรณ์)
$$R_s = \frac{2(t_2 - t_1)}{W_1 + W_2} = \frac{\\sqrt{N}}{4}\left(\frac{\\alpha - 1}{\\alpha}\right)\left(\frac{k'}{k'+1}\right)$$
ทฤษฎีเพลต (Plate Theory) ออก 1 ข้อแน่นอน!
• คอลัมน์ดี = $N$ มาก และ $H$ น้อย
จำนวน Theoretical Plates (N) และ Plate Height (H)
$$N = \frac{L}{H} = 16\left(\frac{t_R}{W_b}\right)^2 = 5.54\left(\frac{t_R}{W_h}\right)^2$$
• $W_b$ = ความกว้างฐานพีค = $4\\sigma$  |  $W_h$ = ความกว้างที่ครึ่งความสูง = $2.354\\sigma$
• $H = \\sigma^2 / L$ — ยิ่ง $H$ น้อย พีคยิ่งแคบ ประสิทธิภาพยิ่งดี
Peak Asymmetry Factor ($A_s = B/A$)
• $A_s$ = 0.9–1.1 → Gaussian peak (ดีที่สุด)
• $A_s$ > 1.1 → Tailing peak — เกิดจาก SP (มี active sites มากเกิน)
• $A_s$ < 0.9 → Fronting peak — เกิดจาก MP (ฉีดสารมากเกินไปหรือ overloading)
• Tailing peak → หา peak area ยาก, บ่งบอกว่าคอลัมน์และระบบการแยกไม่ดี
Van Deemter Equation ออกแน่!
Van Deemter Equation — อธิบายสาเหตุที่ทำให้ H เพิ่มขึ้น
$$H = A + \frac{B}{u} + C_S u + C_M u$$
เทอมชื่อกลไกวิธีลด H
$A$Eddy Diffusionสารเดินทางผ่าน packing ต่างทาง → ออกมาต่างเวลาลดขนาด packing ให้เล็กและสม่ำเสมอ
$B/u$Longitudinal Diffusionสารแพร่จาก $c$ สูง → ต่ำ ตามแนวคอลัมน์เพิ่มอัตราการไหล $u$
$C_S u$SP Mass Transferสารเคลื่อนที่เข้า-ออก SP ช้า ถ้า SP หนาทำ SP ให้บางลง
$C_M u$MP Mass Transferสารไหลผ่านอนุภาค packing ต่างๆ ความเร็วต่างกันลดขนาดอนุภาค / คอลัมน์เล็กลง
• ที่ $u$ ต่ำ: $B/u$ ครอบงำ (longitudinal diffusion สำคัญ) / ที่ $u$ สูง: $Cu$ ครอบงำ (mass transfer สำคัญ)
• $u_{opt}$ = อัตราการไหลที่ $H$ น้อยที่สุด (ได้จากจุดต่ำสุดของกราฟ $H$ vs $u$)
Q&A เน้นข้อสอบ 🔥 ออกแน่
Q: จงนิยาม Chromatography
A: เทคนิคการแยกโดยใช้ความแตกต่างของอัตราการเคลื่อนที่ของแต่ละองค์ประกอบในของผสมบนเฟสคงที่ ภายใต้การพาของเฟสเคลื่อนที่ ได้ผลเป็นโครมาโทแกรม
Q: คอลัมน์ที่ดีควรมี $N$ และ $H$ อย่างไร?
A: $N$ มาก (เพลตมาก) และ $H$ น้อย (เพลตบาง) — $H = L/N$ ดังนั้นคอลัมน์ยาว $N$ มากกว่า แต่ $H$ ไม่จำเป็นต้องน้อยกว่า
Q: $R_s < 1.5$ หมายความว่าอะไร? แก้ยังไง?
A: พีค 2 พีคยังไม่แยกสมบูรณ์ → ปรับ $N$ (เพิ่มความยาว/ลด packing size), $\\alpha$ (เปลี่ยน SP หรือ MP), หรือ $k'$ (ปรับอุณหภูมิหรือ MP composition)
Q: Tailing peak เกิดจากอะไร?
A: $A_s > 1.1$ เกิดจาก stationary phase มี active sites มากเกิน หรือ packing ไม่สม่ำเสมอ → ทำให้ peak area คำนวณยาก
Q: Van Deemter ที่ $u$ ต่ำ ควรลด $H$ ด้วยวิธีใด?
A: ที่ $u$ ต่ำ เทอม $B/u$ (longitudinal diffusion) ครอบงำ → ควรเพิ่มอัตราการไหล $u$ ขึ้น
ตัวอย่างโจทย์คำนวณ 🧮 ฝึกคำนวณ
โจทย์ 1: คอลัมน์ยาว 30 cm, $t_R = 15.0$ min, $W_b = 1.5$ min จงหา N และ H
\[ N = 16\left(\frac{t_R}{W_b}\right)^2 = 16\left(\frac{15.0}{1.5}\right)^2 = 16 \times 100 = 1600 \]
\[ H = \frac{L}{N} = \frac{30}{1600} = 0.01875 \text{ cm} \]
N = 1600 plates, H = 0.01875 cm
โจทย์ 2: สาร A: $t_R = 8.0$ min, สาร B: $t_R = 10.0$ min, $t_M = 1.0$ min, $W_1 = 1.2$ min, $W_2 = 1.4$ min จงหา $k'_A$, $k'_B$, $\alpha$ และ $R_s$
$k'_A = (8.0-1.0)/1.0 = 7.0$    $k'_B = (10.0-1.0)/1.0 = 9.0$
$\alpha = k'_B/k'_A = 9.0/7.0 = 1.29$
\[ R_s = \frac{2(10.0-8.0)}{1.2+1.4} = \frac{4.0}{2.6} = 1.54 \]
$k'_A = 7.0$, $k'_B = 9.0$, $\alpha = 1.29$, $R_s = 1.54$ (แยกสมบูรณ์ เพราะ ≥ 1.5)
โจทย์ 3: $t_R = 12.0$ min, $W_h = 0.80$ min จงหา N
\[ N = 5.54\left(\frac{t_R}{W_h}\right)^2 = 5.54\left(\frac{12.0}{0.80}\right)^2 = 5.54 \times 225 \]
N = 1247 plates
โจทย์ 4: ต้องการ $R_s = 2.0$ จาก $R_s = 1.0$ โดยเพิ่ม N อย่างเดียว ต้องเพิ่ม N กี่เท่า? และต้องยืดคอลัมน์กี่เท่า?
จาก $R_s \propto \sqrt{N}$ → ถ้า $R_s$ เพิ่ม 2 เท่า → N ต้องเพิ่ม $2^2 = 4$ เท่า
และเนื่องจาก $N \propto L$ → ต้องยืดคอลัมน์ 4 เท่าด้วย
เพิ่ม N และ L อีก 4 เท่า
โจทย์ 5 (Peak asymmetry): พีคมี A = 0.8 cm, B = 1.2 cm (วัดที่ 10% ของความสูง) จงหา $A_s$ และระบุว่าเป็น peak แบบใด
$A_s = B/A = 1.2/0.8 = 1.5$
$A_s = 1.5 > 1.1$ → Tailing peak (ปัญหาจาก SP)
Gas Chromatography (GC) คืออะไร?
• เทคนิคที่ต้องเปลี่ยนสถานะสารให้เป็นแก๊สก่อน แล้วพา MP (แก๊ส) ผ่านคอลัมน์ที่มี SP
• ใช้กับสารที่ระเหยได้ (boiling point < 400°C) เท่านั้น
GLC (Gas-Liquid)
• SP = ของเหลวเคลือบบนของแข็ง
• กลไก = Partition ระหว่าง gas–liquid
• ใช้แยกสารได้หลากหลาย (นิยมกว่า)
GSC (Gas-Solid)
• SP = ของแข็ง
• กลไก = Adsorption
• ใช้แยกสารโมเลกุลเล็ก เช่น แก๊ส
ส่วนประกอบเครื่อง GC
1. แก๊สพา (Carrier Gas)
• ต้องไม่ทำปฏิกิริยากับสารตัวอย่าง, การแพร่น้อย, น้ำหนักโมเลกุลต่ำ, บริสุทธิ์สูง
• ควรผ่าน molecular sieve (trap) ก่อนเพื่อกำจัดความชื้น
• ตัวอย่าง: He (sensitivity ดีที่สุดกับ TCD), N₂, H₂, Ar
2. ส่วนฉีดสาร (Injector)
• อุณหภูมิต้องน้อยกว่าอุณหภูมิสูงสุดที่ SP ทนได้ เพื่อไม่ให้ SP สลายตัว
• อุณหภูมิต้องสูงพอให้สารระเหยทันที (biasanya 50°C เหนือ bp ของสาร)

FID — อธิบายหลักการ (ข้อสอบ!)
• FID = Flame Ionization Detector
• ชื่อมีคำว่า "ionization" → มี $e^-$ เกี่ยวข้อง
• เมื่อสารไฮโดรคาร์บอนเข้มข้นสูงถูกเผาในเปลวไฟ H₂/Air → ผลิต ion และ $e^-$ ออกมา
• ยิ่งความเข้มข้นสูง → สัญญาณ (กระแส) ยิ่งมาก
• Selective กับ hydrocarbons เท่านั้น — ไม่ตอบสนองต่อ H₂O, CO₂, SO₂
ECD — หลักการ
• $^{63}$Ni เป็น radioactive source ปล่อย $\\beta$ ray ($e^-$ พลังงานสูง)
• $e^-$ จาก $\\beta$ ray ชน carrier gas → ผลิต $e^-$ อิสระ → วัดได้เป็น baseline current
• เมื่อ halogenated compound เข้ามา → halogen จับ $e^-$ → current ลดลง → peak ปรากฏ
• ยิ่งมี halogen มาก ยิ่ง sensitive
TCD — หลักการ
• วัดการเปลี่ยนแปลงการนำความร้อน (thermal conductivity) ของ gas ที่ออกจากคอลัมน์
• Universal detector — ตรวจวัดได้ทุกสาร (รวม H₂O, CO₂)
• sensitivity ต่ำกว่า FID และ ECD

Q: อธิบายหลักการทำงานของ FID
A: FID = Flame Ionization Detector ชื่อมี ionization → $e^-$ เกี่ยวข้อง เมื่อ HC เข้มข้นสูงถูกเผาในเปลวไฟ H₂/Air → ผลิต ion + $e^-$ → วัดกระแส ยิ่งเข้มข้นมาก สัญญาณยิ่งมาก Selective กับ HC เท่านั้น
Q: SP ไม่มีขั้ว vs มีขั้ว ดูจากอะไร? แยกตามอะไร?

Q: ECD ใช้กับสารอะไร? ทำไม?
A: Halogenated compounds — เพราะ $^{63}$Ni ปล่อย $\\beta$ ray สร้าง $e^-$ อิสระ เมื่อ halogen (มี electronegativity สูง) เข้ามาจับ $e^-$ → current ลด → วัดการลดลงนั้น sensitivity ดีมาก (fg range)
Q: ทำไมต้องผ่านแก๊สพาผ่าน molecular sieve?
A: เพื่อกำจัดความชื้นและ impurities ออกจากแก๊สพา ซึ่งอาจรบกวน detector หรือทำให้ SP เสื่อมสภาพ
Q: Capillary column vs Packed column ต่างกันอย่างไร?
A: Capillary = ยาว 2–50 m, ผนังเคลือบ SP → N สูงกว่า resolution ดีกว่า แต่ sample load น้อย / Packed = ยาว 2–3 m, บรรจุ packing ภายใน → N น้อยกว่า แต่ใส่ sample ได้มากกว่า

Q: Normal phase vs Reversed phase ต่างกันอย่างไร?
A: Normal = SP มีขั้ว MP ไม่มีขั้ว สารไม่มีขั้วออกก่อน / Reversed = SP ไม่มีขั้ว MP มีขั้ว สารมีขั้วออกก่อน — Reversed phase นิยมมากกว่าเพราะใช้ aqueous MP ได้
Q: Isocratic vs Gradient elution ต่างกันอย่างไร?
A: Isocratic = MP เดียวตลอด (ง่ายกว่า ใช้เมื่อ $k'$ ของสารใกล้กัน) / Gradient = เปลี่ยน MP composition ระหว่างรัน (ดีเมื่อสารมี $k'$ ต่างกันมาก หรือ sample ซับซ้อน)
Q: ทำไม RI detector ถึงใช้กับ gradient elution ไม่ได้?
A: RI detector วัด bulk property ของ MP รวมกับ solute เมื่อ MP เปลี่ยน composition → RI ของ MP เองเปลี่ยน → baseline drift → วัดไม่ได้
Q: PDA ดีกว่า Fixed $\\lambda$ detector อย่างไร?
A: PDA วัดได้หลาย $\\lambda$ พร้อมกัน → ได้ full spectrum ของแต่ละ peak → ยืนยัน purity ของ peak ได้ / Fixed วัดได้แค่ $\\lambda$ เดียวที่ตั้งไว้
Q: Guard column มีหน้าที่อะไร? สำคัญอย่างไร?
A: กรองอนุภาคและสารปนเปื้อนก่อนเข้าคอลัมน์หลัก → ยืดอายุคอลัมน์หลักซึ่งราคาแพง — ต้องเปลี่ยน guard column บ่อยกว่าคอลัมน์หลัก

🔴 AAS (Atomic Absorption)
• วัดการดูดกลืนแสงของอะตอม
• ใช้ความร้อน → atomize เป็นอะตอมเสรี
• ใช้ได้กับ 99.98% ของธาตุทั้งหมด
• Beer's Law ใช้ได้ ($A$ vs $c$)
• ต้องมี HCL เป็น light source
• มีความแม่น เที่ยง sensitivity สูง
🟠 AES (Atomic Emission)
• วัดการคายพลังงานของอะตอม
• ใช้ความร้อน → excite → คายพลังงาน
• ใช้ได้เฉพาะอะตอมใน excited state
• Beer's Law ใช้ไม่ได้ ($I$ vs $c$)
• ไม่ต้องมี light source
• Block diagram คล้าย AAS แต่ขาด light source
หลักการ AAS
• วิเคราะห์ได้เฉพาะอะตอมโลหะ — ทำได้ทั้งเชิงคุณภาพและปริมาณ แต่ปริมาณวิเคราะห์ดีกว่า
• แต่ละธาตุมีระดับพลังงานจำเพาะ → ความยาวคลื่นแสงที่ดูดกลืนต่างกัน → ใช้ระบุธาตุได้
• ทำให้ธาตุเป็นอะตอมเสรี (free atom) โดยให้ความร้อน — อุณหภูมิพอเหมาะ ถ้าสูงเกินจะเกิด ionization → error
• $\Delta E$ สูง → กระตุ้นยาก ต้องการอุณหภูมิสูง / $\Delta E$ ต่ำ → กระตุ้นง่าย
• ค่า $A$ แปรผันตรงกับความเข้มข้น → ได้กราฟเส้นตรง (Beer's law)
Boltzmann Equation (AES)
Boltzmann Equation — บอกสัดส่วนอะตอมใน excited state
$$\frac{N_j}{N_0} = \frac{g_j}{g_0} \\cdot e^{-\Delta E / kT}$$
ตัวแปรความหมาย
$N_j$, $N_0$จำนวนอะตอมใน excited state และ ground state ตามลำดับ
$g_j$, $g_0$statistical weight (degeneracy) ของ excited และ ground state
$k$Boltzmann constant = $1.38 \times 10^{-23}$ J/K
$T$อุณหภูมิสัมบูรณ์ (K)
$\Delta E$ผลต่างพลังงานระหว่าง excited และ ground state
• อุณหภูมิ↑ → $N_j/N_0$↑ → signal AES↑
• $\Delta E$ น้อย (ธาตุที่กระตุ้นง่าย) → $N_j/N_0$ สูงกว่า

Atomizer — 3 วิธีทำให้เป็นอะตอมเสรี
1. Flame Atomization
• สารตัวอย่างต้องเป็นสารละลายเนื้อเดียวกัน
• ขั้นตอน: Nebulization (เปลี่ยนเป็น aerosol) → Droplet precipitation (หยดใหญ่ตกไปทิ้ง) → Mixing (ผสมแก๊สเชื้อเพลิง) → Desolvation (ระเหยตัวทำละลาย) → Compound deposition (เกิดอะตอมเสรี)
• ชนิด Nebulizer: Concentric Tube (ไม่หนืด), Cross-flow (เกลือสูง), Fritted-disk, Babington (ความหนืดสูงมาก)
2. Electrothermal / Furnace (Flameless)
• ใช้ graphite tube ให้ความร้อน — sensitivity ดีกว่า flame 10–1000 เท่า ใช้กับ sample ปริมาณน้อย
• 3 ขั้นตอน:
• Drying (~110°C) = ระเหยตัวทำละลาย
• Ashing (350–1200°C) = กำจัดสารอินทรีย์และอนินทรีย์ที่ไม่ต้องการ เหลือแต่สารอนินทรีย์เสถียร
• Atomization (2000–3000°C) = ให้ความร้อนสูงสุด → เกิดอะตอมเสรี, ความสูงพีค ∝ ปริมาณธาตุ
3. Hydride Generation
• เปลี่ยนธาตุให้เป็น hydride gas ก่อน (เช่น AsH₃, SeH₂, SnH₄)
• เหมาะกับ As, Se, Pb, Sn, Bi, Ge, Te ที่ต้องการอุณหภูมิสูงมากในการ atomize

Q&A เน้นข้อสอบ 🔥 ออกแน่
Q: AAS vs AES ต่างกันอย่างไร?
A: AAS = วัดการดูดกลืนแสง ต้องมี HCL, Beer's law ใช้ได้, ใช้กับ 99.98% ของธาตุ / AES = วัดการคายพลังงาน ไม่มี HCL, Beer's law ใช้ไม่ได้, ใช้ได้เฉพาะ excited state atoms
Q: HCL ทำงานอย่างไร?
A: 1) Sputtering — ไอออนของ fill gas (Ne/Ar) ชน cathode ที่เคลือบโลหะ → โลหะหลุดออกมาเป็น vapor  2) Excitation — โลหะ vapor ชนกับ fill gas → กระตุ้น  3) Emission — ปล่อยรังสีที่ความยาวคลื่นจำเพาะของธาตุนั้น
Q: Furnace atomization ดีกว่า Flame อย่างไร?
A: sensitivity ดีกว่า 10–1000 เท่า ใช้กับ sample ปริมาณน้อย (µL) แต่ช้ากว่า และมี memory effect มากกว่า — มี 3 ขั้น: Drying → Ashing → Atomization
Q: ทำไมต้องทำ Ashing stage ก่อน Atomization?
A: เพื่อกำจัดสารอินทรีย์และอนินทรีย์ที่ไม่ต้องการออกก่อน เหลือเฉพาะโลหะที่ต้องการวิเคราะห์ → ลด matrix interference ใน atomization stage
Q: Standard Addition ใช้เมื่อไหร่? แตกต่างจาก Calibration curve อย่างไร?
A: ใช้เมื่อมี matrix interference ที่ทำให้ slope ของ calibration curve เปลี่ยน — Standard Addition เติม known amount ของ standard ลงใน sample โดยตรง → กำจัด matrix effect ได้ → extrapolate กราฟ → หา $c$ ของ sample
Q: EDL เหมาะกับธาตุใด? ทำไม?
A: As, Se, Hg — เพราะธาตุเหล่านี้ต้องการ line intensity สูงมากสำหรับ sensitivity ที่ดี HCL ให้ intensity ไม่พอ แต่ EDL ใช้คลื่นวิทยุ → ให้ intensity สูงกว่า HCL มาก
ตัวอย่างโจทย์คำนวณ 🧮 ฝึกคำนวณ
โจทย์ 1 (Calibration curve): วัด A ของ Fe standard ได้: 0 ppm → A=0.000, 2 ppm → A=0.120, 4 ppm → A=0.240, 6 ppm → A=0.360 จงหาสมการ calibration curve แล้วหา ppm ของ sample ที่ A = 0.210
slope = 0.360/6 = 0.060 A/ppm → A = 0.060 × c
c = 0.210 / 0.060
c = 3.50 ppm
โจทย์ 2 (Standard Addition): Sample มี A = 0.320 เมื่อเติม standard 10 µg/mL ลงใน sample ได้ A = 0.480 จงหา c ของ sample
slope = ΔA/Δc = (0.480 − 0.320)/10 = 0.016 A per µg/mL
c = A(sample)/slope = 0.320/0.016
c = 20 µg/mL
โจทย์ 3 (Standard Addition หลายจุด): Sample A = 0.280, เติม std 5 µg/mL → A = 0.350, เติม std 10 µg/mL → A = 0.420 จงหา c ของ sample
plot A vs c_added → slope = (0.420−0.280)/10 = 0.014, intercept = 0.280
x-intercept (c ของ sample) = −intercept/slope = −0.280/0.014
c = 20 µg/mL
โจทย์ 4: Furnace AAS วัด Pb ใน blood จาก atomization peak height ∝ c ถ้า standard 50 µg/L ให้ peak = 0.480 และ sample ให้ peak = 0.192 จงหา Pb ใน blood
c(sample) = 0.192/0.480 × 50
c = 20 µg/L
โจทย์ 5 (เปรียบเทียบ AAS vs AES): ทำไม AAS ใช้ได้กับธาตุ 99.98% แต่ AES ใช้ได้แค่บางธาตุ?
AAS: ธาตุส่วนใหญ่อยู่ใน ground state → ดูดกลืนแสงได้ถ้าให้แสงที่ถูกความยาวคลื่น → ทำได้เกือบทุกธาตุ
AES: ต้องการธาตุใน excited state → ขึ้นกับอุณหภูมิ flame ตาม Boltzmann → ที่ flame อุณหภูมิปกติมีธาตุใน excited state น้อยมาก → sensitive เฉพาะธาตุที่ $\Delta E$ ต่ำ (กระตุ้นง่าย)
AAS ดีกว่า AES สำหรับ quantitative analysis ส่วนใหญ่
📌 จำสั้น ๆ ก่อนสอบ
• AAS = ดูดกลืน มี HCL Beer's law ใช้ได้ / AES = คายพลังงาน ไม่มี HCL
• HCL: Sputtering → Excitation → Emission (3 ขั้น)
• Furnace: Drying → Ashing → Atomization / sensitivity ดีกว่า flame มาก
• Hydride = As, Se, Pb / EDL = As, Se, Hg / Acetylene-oxygen = ร้อนที่สุด
• Standard Addition = ใช้เมื่อมี matrix interference
Lab 3 · ความกระด้างของน้ำ — EDTA Complexometric Titration
หลักการ
• Complexometric titration = การไทเทรตโดยสร้างสารประกอบเชิงซ้อน (complex) ระหว่าง metal ion กับ chelating agent
• EDTA (Ethylenediaminetetraacetic acid) = chelating agent ที่จับกับ Ca²⁺ และ Mg²⁺ ในอัตราส่วน 1:1 เสมอ ไม่ว่าประจุจะเป็นเท่าไหร่
• ความกระด้างของน้ำ = ปริมาณ Ca²⁺ และ Mg²⁺ ที่ละลายในน้ำ วัดเป็น mg/L CaCO₃
ทำไม pH ต้องเป็น 10?
• ถ้า pH ต่ำ → EDTA จะ protonate → จับกับ Ca²⁺ ได้น้อยลง → ค่าต่ำกว่าจริง
• ถ้า pH สูงเกิน → Ca²⁺ และ Mg²⁺ จะตกตะกอนเป็น Ca(OH)₂ และ Mg(OH)₂ → ค่าต่ำกว่าจริง
• pH 10 ด้วย NH₃/NH₄Cl buffer = จุดที่ EDTA จับ Ca²⁺ และ Mg²⁺ ได้สมบูรณ์ที่สุด
Indicator — Eriochrome Black T (EBT)
• ก่อนจุดยุติ: EBT จับกับ Ca²⁺/Mg²⁺ → สี แดงองุ่น
• จุดยุติ: EDTA แย่ง Ca²⁺/Mg²⁺ จาก EBT → EBT อิสระ → สี น้ำเงิน
• เหตุที่ EDTA แย่งได้: EDTA จับ Ca²⁺ แน่นกว่า EBT (K_f ของ EDTA-Ca > K_f ของ EBT-Ca)
ชนิดความกระด้างและวิธีแยก
Total Hardness
• Ca²⁺ + Mg²⁺ ทั้งหมด
• ไทเทรตน้ำประปาโดยตรง
• = Permanent + Temporary
Permanent Hardness
• Ca²⁺, Mg²⁺ ใน SO₄²⁻, Cl⁻ — ไม่ตกตะกอนเมื่อต้ม
• ต้มน้ำ 20-30 นาที → กรอง → ไทเทรต filtrate
• Temporary = Total − Permanent
• Temporary hardness = Ca(HCO₃)₂ และ Mg(HCO₃)₂ — เมื่อต้ม CO₂ หนีออก → ตกตะกอน CaCO₃ และ Mg(OH)₂
• ทำไมห้ามล้างกระดาษกรอง: ตะกอน CaCO₃ จะละลายกลับ → permanent hardness สูงกว่าจริง → temporary hardness ต่ำกว่าจริง
สูตรคำนวณ
Standardize EDTA ด้วย CaCO₃ primary standard
\[ M_{EDTA} \times V_{EDTA} = M_{CaCO_3} \times V_{CaCO_3} \]
Total / Permanent Hardness (mg/L CaCO₃) — MWCaCO₃ = 100.09 g/mol
\[ \text{Hardness} = \frac{M_{EDTA} \times V_{EDTA} \times 100.09 \times 1000}{V_{sample}\ (\text{mL})} \]
• Temporary Hardness = Total − Permanent
โจทย์คำนวณตัวอย่าง 🧮
จาก lab: MEDTA = 0.00845 M, VEDTA (total) = 11.75 mL, Vsample = 100 mL จงหา Total Hardness
\[ \text{Hardness} = \frac{0.00845 \times 11.75 \times 100.09 \times 1000}{100} \]
Total Hardness ≈ 99.4 mg/L CaCO₃
โจทย์ 2: น้ำ 100 mL ไทเทรตด้วย EDTA 0.01 M ใช้ไป 15.6 mL จงหา Total Hardness
\[ \text{Hardness} = \frac{0.01 \times 15.6 \times 100.09 \times 1000}{100} \]
Total Hardness = 156.1 mg/L CaCO₃
โจทย์ 3: Total = 156.1, Permanent = 89.3 mg/L CaCO₃ จงหา Temporary Hardness
Temporary = Total − Permanent = 156.1 − 89.3
Temporary Hardness = 66.8 mg/L CaCO₃
Q&A เน้นข้อสอบ 🔥
Q: ทำไม EDTA จับ metal ion ในอัตรา 1:1 เสมอ?
A: EDTA เป็น hexadentate ligand — มีตำแหน่งให้ประจุ 6 ตำแหน่ง (N 2 และ O 4) จับล้อมรอบ metal ion หมดพอดี 1 โมเลกุล ไม่ว่า charge ของ metal จะเป็น +2 หรือ +3
Q: ทำไม pH ต้องเป็น 10 ไม่ใช่ 7 หรือ 12?
A: pH 7 = EDTA protonate มากเกิน จับ Ca²⁺ ได้น้อย / pH 12 = Ca²⁺, Mg²⁺ ตกตะกอน → ค่าต่ำกว่าจริง / pH 10 = จุดที่ทั้ง EDTA พร้อมจับ และ Ca²⁺, Mg²⁺ ยังละลายอยู่
Q: ทำไม EBT ถึงเปลี่ยนสีได้?
A: EBT จับ Ca²⁺ → complex สีแดงองุ่น เมื่อ EDTA มีมากพอ EDTA แย่ง Ca²⁺ ออกจาก EBT เพราะ K_f(EDTA-Ca) > K_f(EBT-Ca) → EBT อิสระ → สีน้ำเงิน
Q: ทำไมต้องใช้ CaCO₃ เป็น primary standard?
A: CaCO₃ บริสุทธิ์สูง, เสถียร, ไม่ดูดความชื้น, MW แน่นอน → standardize EDTA ได้แม่นยำ
📌 จำสั้น ๆ ก่อนสอบ
• EDTA : Ca²⁺ = 1:1 เสมอ (hexadentate) / pH 10 ด้วย NH₃ buffer
• EBT: แดงองุ่น (จับ Ca²⁺) → น้ำเงิน (EDTA แย่ง Ca²⁺) = จุดยุติ
• Temporary = Total − Permanent / ห้ามล้างกระดาษกรอง!
• CaCO₃ → Hardness หน่วย mg/L CaCO₃
Lab 4 · ไอโอดีนในเกลือ — Iodometric Back-Titration
หลักการ
• Iodometry = การวิเคราะห์โดยใช้ I₂ หรือสารที่สร้าง I₂ ได้เป็นสารตรวจวัด
• ใช้ back-titration: IO₃⁻ (iodate) ในเกลือทำปฏิกิริยากับ KI ที่เติมเกิน → ได้ I₂ → ไทเทรต I₂ ที่ได้ด้วย Na₂S₂O₃
• ทำไมไม่วัด IO₃⁻ โดยตรง? — IO₃⁻ ไม่มีสีและไม่มี indicator ที่เหมาะสม แต่ I₂ ที่ได้สามารถวัดกับน้ำแป้งได้ชัดเจน
ทำไมต้องเติม H₂SO₄?
• ปฏิกิริยาต้องการ H⁺: IO₃⁻ + 5I⁻ + 6H⁺ → 3I₂ + 3H₂O
• ถ้าไม่มี H⁺ ปฏิกิริยาจะไม่เกิด → ไม่ได้ I₂ → วัดไม่ได้
Indicator — น้ำแป้ง (Starch)
• I₂ + starch → complex สี น้ำเงินเข้ม
• จุดยุติ: I₂ หมด → สีน้ำเงินหายไป → ไม่มีสี
• ทำไมเติมน้ำแป้งใกล้จุดยุติ? — ถ้าเติมตั้งแต่ต้น I₂ เข้มข้นสูงจับแป้งแน่นมาก → complex สลายยาก → endpoint ไม่ชัด → error สูง
Stoichiometry ที่ต้องจำ
ปฏิกิริยาที่ 1 — สร้าง I₂
\[ IO_3^- + 5I^- + 6H^+ \rightarrow 3I_2 + 3H_2O \]
ปฏิกิริยาที่ 2 — ไทเทรต I₂
\[ I_2 + 2S_2O_3^{2-} \rightarrow 2I^- + S_4O_6^{2-} \]
สรุป Stoichiometry (ต้องจำ!)
\[ IO_3^- : I_2 : Na_2S_2O_3 = 1 : 3 : 6 \]
• ดังนั้น: $M_{Na_2S_2O_3} \times V_{Na_2S_2O_3} = 6 \times M_{KIO_3} \times V_{KIO_3}$
สูตรคำนวณ
mg ของ I⁻ ในตัวอย่าง (MW ของ I = 126.9 g/mol)
\[ \text{mg I} = \frac{1}{2} \times M_{Na_2S_2O_3} \times V_{Na_2S_2O_3}\ (\text{L}) \times 126.9 \times 1000 \]
mg I⁻ ต่อ kg เกลือ
\[ \text{mg I/kg salt} = \frac{\text{mg I}}{\text{wt sample (g)}} \times 1000 \]
• หาร 2 เพราะ I₂ 1 mol มี I 2 อะตอม แต่เราต้องการ I อะตอมเดี่ยว (I⁻ ในเกลือ)
โจทย์คำนวณตัวอย่าง 🧮

mg I = ½ × 2.012×10⁻³ × 0.01625 × 126.9 × 1000 = 0.0691 mg
mg I/kg = 0.0691 / 1.9958 × 1000
≈ 34.6 mg I/kg salt

$M_{Na_2S_2O_3} = 2.083 \times 10^{-3}$ M
โจทย์ 3: ไทเทรตเกลือ 1.9958 g ด้วย Na₂S₂O₃ 2.083×10⁻³ M ใช้ 16.5 mL จงหา mg I/kg salt

Lab 6 · ไนเตรตด้วย Ion Selective Electrode (ISE) — Potentiometry
หลักการ Potentiometry
• Potentiometry = วิธีวิเคราะห์ที่วัด potential (แรงดันไฟฟ้า, mV) ของ electrochemical cell
• ไม่ผ่านกระแสไฟฟ้าเข้าสารละลาย (วัดแบบ passive) → ไม่รบกวน sample
• ประกอบด้วย 2 ขั้ว: ISE (indicator electrode) + Reference electrode (Ag/AgCl)
Ion Selective Electrode (ISE) คืออะไร?
• เยื่อเลือกผ่าน (selective membrane) ที่ผ่านเฉพาะ ion ที่ต้องการ (เช่น NO₃⁻)
• เมื่อ NO₃⁻ เข้าสู่เยื่อ → เกิด potential ต่างกันระหว่างสองด้าน → วัดเป็น mV
• ยิ่ง NO₃⁻ เข้มข้น → potential ต่างมาก → สัญญาณมาก
Nernst Equation — หัวใจของ ISE
Nernst Equation (สำหรับ anion เช่น NO₃⁻)
\[ E = E^0 - \frac{0.0592}{n} \log[NO_3^-] \]
• E = potential ที่วัดได้ (mV) / n = charge ของ ion (NO₃⁻ → n=1)
• plot E vs log[NO₃⁻] → เส้นตรง → slope ≈ −59.2 mV/decade สำหรับ anion monovalent
• เมื่อรู้สมการเส้นตรง → แทนค่า E ของ sample → หา [NO₃⁻]
ทำไมต้องเติม Ionic Strength Adjustment Buffer (ISAB)?
• Nernst equation จริงๆ ใช้ activity ไม่ใช่ concentration: $a = \gamma \times c$
• ISAB ทำให้ ionic strength ของทุก sample เท่ากัน → activity coefficient (γ) คงที่ → activity ∝ concentration → calibration curve เป็นเส้นตรง
การเตรียมและวัด
1
แช่ ISE ใน DI water ก่อนใช้ — hydrate เยื่อให้สมบูรณ์ ถ้าไม่แช่ reading ไม่เสถียร
2
เตรียม standard NO₃⁻ 1, 5, 10, 20 ppm + ISAB อย่างละ 10 mL → วัด E (mV) ทีละค่า
3
ล้าง DI water ระหว่างทุก sample — ป้องกัน carry-over
4
plot E vs log[NO₃⁻] → เส้นตรง → หา slope และ intercept → แทนค่า E ของ unknown
โจทย์คำนวณตัวอย่าง 🧮
จาก lab: standard 1 ppm → E = 344.0 mV, 20 ppm → E = 243.3 mV จงหา slope ของ calibration curve
slope = ΔE / Δ(log c) = (243.3 − 344.0) / (log20 − log1) = −100.7 / 1.301
slope ≈ −77.4 mV/decade (ควรใกล้ −59.2 mV/decade ตาม Nernst)
โจทย์ 2: ISE วัด NO₃⁻ standard ได้: 1 ppm → 344 mV, 10 ppm → 270 mV จงหา slope และ c ของ sample ที่ E = 296 mV
slope = ΔE/Δ(log c) = (270−344)/(log10−log1) = −74/1 = −74 mV/decade
สมการ: E = 344 − 74×log[c] → 296 = 344 − 74×log[c] → log[c] = (344−296)/74 = 0.649
c = 10^0.649 = 4.46 ppm NO₃⁻
โจทย์ 3: ISE slope วัดได้ −74 mV/decade แต่ theoretical Nernst slope = −59.2 mV/decade ควรทำอย่างไร?
ใช้ slope จากการ calibrate จริง (−74) ไม่ใช่ theoretical ในการคำนวณ เพราะ membrane อาจไม่ ideal 100%
ใช้ empirical slope จาก calibration curve เสมอ
Q&A เน้นข้อสอบ 🔥
Q: ISE ต่างจาก UV-Vis ยังไง?
A: ISE วัด potential (ไฟฟ้า) ตาม Nernst equation → ไม่ต้องมีสาร สี หรือ complex / UV-Vis วัดการดูดกลืนแสงตาม Beer's Law → ต้องมีสารที่ดูดกลืนแสงได้
Q: ทำไม slope ของกราฟ E vs log[c] ต้องเป็น −59.2 mV/decade?
A: จาก Nernst equation: slope = −0.0592/n × 1000 (ถ้า E เป็น mV) สำหรับ monovalent anion n=1 → slope = −59.2 mV/decade ถ้า slope ต่างมาก = ISE ชำรุดหรือเยื่อเสื่อม
Q: ทำไม ISE ต้องแช่น้ำก่อนใช้?
A: เยื่อเลือกผ่านต้องดูดซับน้ำ (hydration) ก่อนจึงจะทำงานได้ ถ้าเยื่อแห้ง → ion ผ่านไม่สม่ำเสมอ → reading drift ไม่เสถียร
Q: ทำไม E ลดลงเมื่อ [NO₃⁻] เพิ่ม (สำหรับ anion)?
A: จาก Nernst: E = E⁰ − (0.0592/n)log[NO₃⁻] → เมื่อ [NO₃⁻] เพิ่ม log[NO₃⁻] เพิ่ม → E ลด (เพราะติดลบ) → กราฟ slope ลบ
📌 จำสั้น ๆ ก่อนสอบ
• Potentiometry วัด mV → ตาม Nernst: E vs log[c] = เส้นตรง slope ≈ −59.2 mV/decade
• ต้องเติม ISAB เพื่อ fix ionic strength → γ คงที่
• แช่ ISE ก่อนใช้ / ล้าง DI ระหว่าง sample
Lab 7 · Conductometric Titration — วัดการนำไฟฟ้า
หลักการ Conductometry
• Conductance (G) = ความสามารถของสารละลายในการนำไฟฟ้า หน่วย Siemens (S)
• G แปรผันตรงกับ จำนวนและ mobility ของ ion ในสารละลาย
• H⁺ และ OH⁻ มี mobility สูงสุด (Grotthuss mechanism — ถ่ายโปรตอนผ่าน H-bond chain ของน้ำ) → G สูงสุด
• ดังนั้นเมื่อ H⁺ หรือ OH⁻ ถูกแทนที่ด้วย ion อื่น → G จะเปลี่ยนแปลงอย่างชัดเจน
ข้อดีเหนือ regular titration
• ใช้กับ สารละลายสีหรือขุ่น ที่ indicator ทั่วไปใช้ไม่ได้
• ไม่ต้องใช้ indicator → ไม่มี error จาก indicator
• หาจุดยุติจาก กราฟ → แม่นยำกว่าดูสีด้วยตา
รูปแบบกราฟ G vs V และ ทำไม?
HCl + NaOH → กราฟ V
• ก่อน EP: H⁺ (G สูง) ถูกแทนด้วย Na⁺ (G ต่ำ) → G ลด
• ที่ EP: H⁺ หมด, Cl⁻ + Na⁺ เหลือ → G ต่ำสุด
• หลัง EP: OH⁻ (G สูง) เพิ่ม → G เพิ่ม
• จุดยุติ = จุดต่ำสุดของกราฟ V
CH₃COOH + NaOH → กราฟ L
• ก่อน EP: CH₃COOH แตกตัวน้อย (weak acid) G ต่ำ → เกิด CH₃COONa (buffer) → G ค่อยๆ เพิ่ม
• หลัง EP: OH⁻ เพิ่มชัน → G เพิ่มเร็ว
• จุดยุติ = จุดเปลี่ยนความชันของกราฟ L
สูตรคำนวณ
Standardize NaOH ด้วย KHP (MW = 204.22 g/mol)
\[ M_{NaOH} = \frac{m_{KHP}\ (\text{g})}{204.22 \times V_{NaOH}\ (\text{L})} \]
หา c ของกรดตัวอย่าง (อ่านจุดยุติจากกราฟ)
\[ c_{acid} \times V_{acid} = c_{NaOH} \times V_{NaOH} \]
• KHP = Potassium Hydrogen Phthalate → primary standard สำหรับ NaOH
• ทำไมใช้ KHP? — pure, stable, MW สูง (ชั่งแม่นยำ), ไม่ดูด CO₂
Q&A เน้นข้อสอบ 🔥
Q: ทำไม H⁺ และ OH⁻ ถึงมี conductance สูงกว่า ion อื่น?
A: Grotthuss mechanism — H⁺ ไม่ได้เคลื่อนที่จริงๆ แต่ถ่ายโปรตอนผ่าน network ของ H-bond ในน้ำได้รวดเร็ว (proton hopping) → effective mobility สูงกว่า Na⁺, K⁺ มาก
Q: ทำไมกราฟ HCl + NaOH เป็น V แต่ CH₃COOH + NaOH เป็น L?
A: HCl = strong acid แตกตัว 100% → H⁺ เยอะ G สูง เมื่อ NaOH เข้า H⁺ ถูกแทนด้วย Na⁺ ที่ G ต่ำกว่า → G ลด / CH₃COOH = weak acid แตกตัวน้อย G เริ่มต้นต่ำ + เกิด buffer เมื่อเติม NaOH → G ค่อยๆ เพิ่ม
Q: อ่านจุดยุติจากกราฟ Conductometric ยังไง?
A: ลากเส้นตรงผ่านจุดก่อน EP และหลัง EP แยกกัน → จุดตัดของ 2 เส้นตรง = จุดยุติ (endpoint)
ตัวอย่างโจทย์คำนวณ 🧮 ฝึกคำนวณ
โจทย์ 1: ชั่ง KHP 2.0131 g ละลายในน้ำ ไทเทรตด้วย NaOH ใช้ไป 10.5 mL จงหา M ของ NaOH (MW KHP = 204.22 g/mol)
\[ M_{NaOH} = \frac{2.0131}{204.22 \times 0.0105} = \frac{2.0131}{2.1443} \]
$M_{NaOH} = 0.0939$ M
โจทย์ 2: จาก conductometric titration ของ HCl 25.0 mL ด้วย NaOH 0.0939 M อ่านจุดยุติจากกราฟได้ที่ V = 2.80 mL จงหา M ของ HCl
\[ c_{HCl} \times 25.0 = 0.0939 \times 2.80 \]
$c_{HCl} = 0.0939 \times 2.80 / 25.0$
$M_{HCl} = 0.01052$ M ≈ 0.0105 M
โจทย์ 3: ทำไม conductometric titration ถึงหา EP ได้แม้สีสารละลายขุ่น?
Conductometry วัด G ทางไฟฟ้า ไม่พึ่งสี → สีของสารละลายไม่รบกวนการวัด
วัด G ทางไฟฟ้า → ไม่ต้องอาศัยการมองสี → ใช้ได้กับสารละลายสีหรือขุ่น
โจทย์ 4: กราฟ conductometric titration ของ CH₃COOH + NaOH จะมีรูปร่างแตกต่างจาก HCl + NaOH อย่างไร?
HCl (strong acid): G เริ่มสูง (H⁺ เยอะ) → ลดก่อน EP → เพิ่มหลัง EP = V shape
CH₃COOH (weak acid): G เริ่มต่ำ (แตกตัวน้อย) → ค่อยๆ เพิ่มก่อน EP (เกิด buffer) → เพิ่มชันหลัง EP = L shape
HCl = V shape / CH₃COOH = L shape
📌 จำสั้น ๆ ก่อนสอบ
• G วัดแทน indicator / H⁺, OH⁻ มี G สูงสุด (Grotthuss mechanism)
• HCl + NaOH = V shape (G ลดก่อน EP) / CH₃COOH + NaOH = L shape
• จุดยุติ = จุดตัดของ 2 เส้นตรงในกราฟ
• KHP = primary standard สำหรับ standardize NaOH
Lab 8 · เหล็กในน้ำ — UV-Vis Spectrophotometry (Beer's Law Application)
หลักการ — เชื่อมกับ UV-Vis tab
• ใช้ Beer's Law: A = εbc → A แปรผันตรงกับ cFe → วัด A แล้ว interpolate จาก calibration curve
• Fe²⁺ + 1,10-phenanthroline (3 โมเลกุล) → [Fe(phen)₃]²⁺ สีแดงส้ม → ดูดกลืนแสง ~510 nm
• ทำไมต้องสร้าง complex? — Fe²⁺ เองไม่มีสี ดูดกลืนแสงในช่วง visible ไม่ได้ → ต้องสร้าง colored complex ก่อนจึงวัด A ได้
ทำไมต้องเติม NH₂OH·HCl ก่อน?
• น้ำในธรรมชาติมีทั้ง Fe²⁺ และ Fe³⁺
• 1,10-phen จับได้ เฉพาะ Fe²⁺ เท่านั้น (ไม่จับ Fe³⁺)
• NH₂OH·HCl = reducing agent → reduce Fe³⁺ → Fe²⁺ ทั้งหมดก่อน → วัดได้ total iron
• ถ้าไม่ reduce → วัดได้แค่ Fe²⁺ ที่มีอยู่แล้ว → ค่าต่ำกว่า total iron จริง
ทำไมต้อง control pH?
• pH ที่เหมาะสม = 3.5–4.5 (ใช้ sodium citrate ปรับ)
• pH ต่ำเกิน (กรด) → 1,10-phen protonated → จับ Fe²⁺ ได้น้อย
• pH สูงเกิน (ด่าง) → Fe²⁺ ตกตะกอนเป็น Fe(OH)₂ → ไม่มี Fe²⁺ ในสารละลาย
• ตรวจ pH ด้วย bromophenol blue: เหลือง (pH ต่ำ) → น้ำเงิน (pH ได้ที่)
ทำไมต้องหัก Blank?
• Blank = สารละลายที่มี reagent ทุกอย่าง ยกเว้น Fe
• A ของ blank = การดูดกลืนแสงจาก reagent เอง (background absorption)
• A จริงของ Fe = A(sample) − A(blank)
• ถ้าไม่หัก blank → A สูงกว่าจริง → cFe สูงกว่าจริง
โจทย์คำนวณตัวอย่าง 🧮
จาก lab: A(blank) = 0.1864, A(sample) = 0.4136 จาก calibration curve: A = 0.800×c + 0.188 (A vs mg Fe/mL) จงหา cFe ใน sample
A จริง = 0.4136 − 0.1864 = 0.2272
0.2272 = 0.800×c + 0.188 → c = (0.2272 − 0.188) / 0.800
cFe ≈ 0.049 mg Fe/mL → แปลงเป็น mg/L คูณ 1000
โจทย์ 2: Calibration curve ของ Fe: A = 0.800×c + 0.000 (c ใน mg Fe/25mL) วัด A(sample) = 0.413, A(blank) = 0.186 จงหา c ของ Fe
A จริง = 0.413 − 0.186 = 0.227
0.227 = 0.800 × c → c = 0.284 mg/25mL
cFe = 0.284 mg/25 mL → ถ้า sample ปริมาตร 1.25 mL → mg/L = 0.284/1.25 × 1000 = 227 mg/L
โจทย์ 3: ทำไม A ของ calibration ต้องอยู่ในช่วง 0.1–1.0?
A < 0.1: T > 79% → noise ของ detector คิดเป็น % error สูง → ไม่แม่นยำ
A > 1.0: Beer's Law เริ่มเบี่ยงเบน (สารเข้มข้นเกิน) → กราฟไม่เป็นเส้นตรง → ค่าผิดพลาด
A = 0.1–1.0 คือช่วงที่ Beer's Law เป็นเส้นตรงและ detector มี S/N ดี
Q&A เน้นข้อสอบ 🔥
Q: Lab นี้ใช้ Beer's Law อย่างไร?
A: สร้าง [Fe(phen)₃]²⁺ ที่มีสี → A ∝ cFe ที่ 510 nm → plot calibration curve (A vs cFe) → เส้นตรง → หา c ของ sample จากกราฟ

Lab 9 · Ethanol ในสุรา — GC-FID + Internal Standard Method
หลักการ — เชื่อมกับ GC tab
• ใช้ GC-FID แยกและวัดปริมาณ ethanol ในสุรากลั่น
• FID เหมาะเพราะ ethanol = organic compound (hydrocarbon) → FID sensitive มาก
• ใช้ Internal Standard (IS) Method แทน External Standard เพราะ injection volume ใน GC ไม่สม่ำเสมอ
Internal Standard Method คืออะไร? ทำไมดีกว่า?
• เติม IS (acetonitrile) ปริมาณ คงที่เหมือนกันทุก sample
• วัด ratio = peak area ethanol / peak area IS แทนที่จะวัด peak area ethanol เดี่ยวๆ
• ถ้าฉีดสารมากกว่าที่ควร → peak area ethanol ↑ แต่ peak area IS ↑ ด้วย → ratio คงที่ → error หักล้างกัน
• ทำไมใช้ Acetonitrile เป็น IS? — ไม่มีใน sample สุรา, elute ที่เวลาใกล้กับ ethanol แต่แยกพีคชัดเจน, FID ตรวจจับได้ดี
ทำไมต้องสกัดด้วย 1-Octanol ก่อน?
• สุรามีน้ำเป็นส่วนใหญ่ → น้ำทำลาย GC column และ detector
• 1-Octanol สกัด ethanol ออกจาก aqueous phase → ได้ organic layer ที่มี ethanol + IS
• ฉีดเฉพาะชั้น Octanol เข้าเครื่อง → ลด matrix interference จากน้ำ

สูตรและ Calibration
Calibration Curve — Internal Standard Method
\[ y = \frac{A_{ethanol}}{A_{IS}} \quad \text{vs} \quad x = \%\ \text{ethanol (v/v)} \]
• หาสมการเส้นตรง y = mx + b → แทน y ของ sample → หา %ethanol
• R² ควรใกล้ 1.0 = linearity ดี
โจทย์คำนวณตัวอย่าง 🧮
จาก lab: calibration ได้สมการ y = 0.0596x + 0.0 (ratio vs %ethanol), sample มี ratio = 1.204 จงหา %ethanol ในสุรา
1.204 = 0.0596x + 0.0 → x = 1.204 / 0.0596
%ethanol ≈ 20.2% v/v
โจทย์ 2: Calibration: 10% → ratio 0.547, 20% → 1.126, 30% → 1.738 จงหาสมการ calibration curve
slope = (1.738−0.547)/(30−10) = 1.191/20 = 0.05955 per %ethanol
intercept ≈ 0.547 − 0.05955×10 = −0.049 ≈ 0 (ผ่าน origin)
ratio ≈ 0.0596 × %ethanol (R² ≈ 1.0)
โจทย์ 3: ถ้าฉีด sample โดยไม่มี IS และ injection volume เปลี่ยนจาก 1.0 µL เป็น 0.8 µL peak area ethanol ลด 20% ผลกระทบต่อค่าที่รายงาน?
ถ้าไม่มี IS: peak area ลด 20% → รายงาน %ethanol ต่ำกว่าจริง 20% → error มาก
ถ้ามี IS: peak area IS ก็ลด 20% เช่นกัน → ratio คงที่ → ไม่มี error
IS ชดเชย injection variability → ผลไม่เปลี่ยน
Q&A เน้นข้อสอบ 🔥
Q: Internal standard method ดีกว่า external standard อย่างไร?

Lab 10 · พาราเซตามอลในยา — Reversed Phase HPLC
หลักการ — เชื่อมกับ HPLC tab
• ใช้ Reversed Phase HPLC: SP = C18 (ไม่มีขั้ว), MP = MeOH:H₂O 50:50 (มีขั้ว)
• พาราเซตามอลมีขั้วปานกลาง (มี –OH และ –NHCOCH₃) → ถูก retain บน C18 ในระดับที่เหมาะ → elute ออกมาในเวลาที่วัดได้
• วัดด้วย UV 254 nm เพราะพาราเซตามอลมี aromatic ring (benzene) → π→π* transition → ดูดกลืนแสง UV ที่ 254 nm ได้ดี
ทำไมต้องกรอง membrane 0.45 µm?
• ยาเม็ดมี excipients (แป้ง, binding agent) ที่ไม่ละลายหมด
• อนุภาคเล็กอุดตัน column C18 ที่ราคาแพง → column เสียหาย → ต้องกรองออกก่อน
• Guard column ยังช่วยกรองอีกชั้น แต่ membrane filter ช่วยยืดอายุ guard column ด้วย
ทำไม MeOH:H₂O 50:50?
• ต้องการ k' ที่เหมาะสม (k' = 2–10 สำหรับ separation ที่ดี)
• MeOH มาก → MP ไม่มีขั้วมากขึ้น → k' ลด → paracetamol ออกเร็ว (อาจเร็วเกินไป overlap กับ peak อื่น)
• น้ำมาก → MP มีขั้วมากขึ้น → k' เพิ่ม → paracetamol ออกช้า (ใช้เวลานาน)
• 50:50 = จุดที่ balance ระหว่าง resolution ดีและเวลาสมเหตุสมผล
การคำนวณปริมาณพาราเซตามอล
จาก calibration curve (peak area vs concentration) → หา c ของ sample
\[ \text{mg paracetamol/tablet} = c_{sample} \times \frac{V_{total}}{m_{weighed}} \times m_{tablet} \]
• csample = ความเข้มข้นที่อ่านจากกราฟ (mg/mL)
• Vtotal = ปริมาตรที่ปรับไว้ (25 mL)
• mweighed = น้ำหนักยาที่ชั่งวิเคราะห์ (g)
• mtablet = น้ำหนักยาทั้งเม็ด (g)
โจทย์คำนวณตัวอย่าง 🧮
จาก lab: csample (จากกราฟ) = 0.382 mg/mL, Vtotal = 25 mL, mweighed = 0.0244 g, mtablet = 0.6248 g จงหา mg paracetamol/tablet
mg para = 0.382 × 25 = 9.55 mg (ในสารละลายที่ชั่ง 0.0244 g)
mg para/tablet = 9.55 × (0.6248 / 0.0244)
≈ 244.5 mg/tablet (ยาพาราเซตามอลปกติมี 500 mg/tablet)
โจทย์ 2: Calibration curve: peak area = 12500 × c (µg/mL) วัด sample ได้ peak area = 4,850 จงหา c
c = 4850 / 12500
c = 0.388 µg/mL = 0.388 mg/L
โจทย์ 3: จาก โจทย์ 2: c = 0.388 mg/mL ใน 25 mL, ชั่งยา 0.0244 g จาก tablet 0.6248 g จงหา mg paracetamol/tablet
mg ใน 25 mL = 0.388 × 25 = 9.70 mg (จาก 0.0244 g tablet)
mg/tablet = 9.70 × (0.6248/0.0244)
≈ 248 mg/tablet
โจทย์ 4: ถ้าเปลี่ยน MP จาก MeOH:H₂O 50:50 เป็น 70:30 paracetamol จะออกเร็วหรือช้ากว่า?
MeOH เพิ่ม → MP ไม่มีขั้วมากขึ้น → C18 และ MP มีขั้วใกล้กัน → k' ลด → paracetamol ออกเร็วขึ้น

Q&A เน้นข้อสอบ 🔥
Q: ทำไม lab นี้ใช้ reversed phase ไม่ใช่ normal phase?
A: พาราเซตามอลมีขั้วปานกลาง → Normal phase (SP มีขั้ว) จะ retain แน่นเกิน ออกยาก / Reversed phase (C18, ไม่มีขั้ว) + MeOH/water → k' เหมาะสม elute ออกมาในเวลาที่วัดได้สะดวก
Q: ทำไมวัดที่ 254 nm ไม่ใช่ 510 nm แบบ Lab 8?
A: พาราเซตามอลมี benzene ring → UV absorption ที่ ~254 nm (π→π*) ไม่มีสีในช่วง visible / Lab 8 วัด visible (510 nm) เพราะ [Fe(phen)₃]²⁺ มีสีแดงส้ม
Q: ถ้าลด MeOH เหลือ 30% จะเกิดอะไรกับ chromatogram?
A: MP มีขั้วมากขึ้น → k' เพิ่ม → paracetamol ติด C18 นานขึ้น → tR เพิ่ม → peak ออกช้าและกว้างขึ้น → ใช้เวลา run นานขึ้น
Q: ต่างกับ Lab 9 (GC) อย่างไร?
A: HPLC ใช้ liquid MP → ใช้กับสารที่ระเหยยากหรือ thermally labile ได้ เช่น ยา / GC ต้องเปลี่ยนสารเป็นแก๊ส → ใช้กับสารระเหยง่ายเท่านั้น เช่น ethanol
📌 จำสั้น ๆ ก่อนสอบ
• C18 (reversed phase) + MeOH:H₂O 50:50 + UV 254 nm (π→π* ของ benzene ring)
• กรอง membrane 0.45 µm ก่อนฉีด ป้องกัน column อุดตัน
• MeOH↑ → k'↓ สารออกเร็ว / น้ำ↑ → k'↑ สารออกช้า
• HPLC = สารระเหยยาก / GC = สารระเหยง่าย
